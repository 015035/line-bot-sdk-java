# line-bot-api-client

LINE bot API client for Java 8.

## SYNOPSIS

        DefaultLineBotClient defaultLineBotClient = new DefaultLineBotClient(
                "YOUR_CHANNEL_ID",
                "YOUR_CHANNEL_SECRET",
                "YOUR_CHANNEL_MID");
        defaultLineBotClient.sendText("USER_MID", "Hello");

## DESCRIPTION

This library provides API client for LINE bot API.
